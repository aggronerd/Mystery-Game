Detailed Description

An ApplicationModule represents part of the overall application where all inherent classes would not expect to be instantiated at the same time but share common components, such as a window manager, gui object, graphics context, input slots, a "run" function to initiate the loop and a class variable for breaking the loop.

This ensures that the editor and the game do not have to repeat this common code and instead inherit this object.
